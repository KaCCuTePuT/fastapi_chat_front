## Пробный чат на fastapi

***

Фронтенд для аналога Телеграмма.

* Регистрация по смс, авторизация
* Чат на вебсокетах

* Добавление, удаление друзей
* Исправление недочетов

### Используемый стек на данный момент:
* Vue
* Fastapi (в другом репозитории)
